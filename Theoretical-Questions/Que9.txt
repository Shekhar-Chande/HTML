Que:- . Give an example of an HTML tag with attributes.

Ans:- 

The href Attribute
The <a href="www.google.com"> tag defines a hyperlink. The href attribute specifies the URL of the page the link .

The src Attribute
The <img src="img_girl.jpg"> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed

The width and height Attributes
The <img src="img_girl.jpg" width="200" height="400"> tag should also contain the width and height attributes, which specify the width and height of the image 

The alt Attribute
The required alt attribute for the <img> tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to a slow connection, or an error in the src attribute, or if the user uses a screen reader. 

<img src="img_girl.jpg" alt="Girl with a jacket">
